Basic Git Information

Git is a Distributed Version Control System (DVCS) used to track changes in source code and collaborate with developers efficiently.

Key Features

✅ Tracks file changes over time
✅ Maintains version history
✅ Supports team collaboration
✅ Enables branching and merging
✅ Allows rollback to previous versions

What is Git?

Git is a Distributed Version Control System (DVCS) used to track changes in files and source code. It helps developers manage projects, collaborate with teams, and maintain a history of changes.

Simple Definition

Git is a tool that records changes in your files so you can track, manage, and restore previous versions whenever needed.

Create and manage branches
Merge changes safely

Basic Git Commands & Working with GitHub Repositories
1. Configure Git (First Time)
git config --global user.name "Your Name"
git config --global user.email "your@email.com"
2. Create a New Repository
git init
3. Check Repository Status
git status
4. Add Files to Staging Area
git add filename.txt

Add all files:

git add .
5. Commit Changes
git commit -m "Initial commit"
6. View Commit History
git log
7. Clone a GitHub Repository
git clone <repository-url>

Example:

git clone https://github.com/username/repository.git
8. Connect Local Repository to GitHub
git remote add origin <repository-url>
9. Push Code to GitHub
git push -u origin main
10. Pull Latest Changes
git pull origin main
11. Create a New Branch
git branch feature-branch
12. Switch Branch
git checkout feature-branch

Or create and switch together:

git checkout -b feature-branch
13. Merge Branch
git merge feature-branch
